## Phase 3 — Backend Evolution Client

Create a backend helper for all Evolution API calls.

File suggestion:

```text
src/services/evolutionClient.ts
```

Responsibilities:

- [ ] Add authenticated request wrapper using `apikey` header.
- [ ] Create instance.
- [ ] Connect instance / fetch QR code.
- [ ] Check connection state.
- [ ] Send text message.
- [ ] Logout/delete/disconnect instance.
- [ ] Configure webhook.

Example helper shape:

```ts
export async function evolutionRequest(path: string, options = {}) {
  const url = `${process.env.EVOLUTION_API_URL}${path}`;

  const response = await fetch(url, {
    ...options,
    headers: {
      "Content-Type": "application/json",
      "apikey": process.env.EVOLUTION_API_KEY!,
      ...(options as any).headers,
    },
  });

  const data = await response.json().catch(() => null);

  if (!response.ok) {
    throw new Error(`Evolution API error ${response.status}: ${JSON.stringify(data)}`);
  }

  return data;
}
```
